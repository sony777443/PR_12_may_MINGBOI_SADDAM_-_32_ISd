# Новости

Простой статический веб-проект на HTML/CSS/JS.

## О проекте

Это одностраничный новостной сайт с тёмной темой и ссылками на дополнительные материалы.
Проект реализован без сборщиков и фреймворков, только чистый HTML, встроенные стили и JavaScript.

## Файлы

- `index.html` — главная страница с новостными карточками и переключателем темы.
- `2index.html`, `3index.html`, `4index.html` — дополнительные страницы для перехода по ссылкам.
- `фото1.png`, `фото2.png`, `фото4.png`, `3.png` — изображения для контента.
- `README.md` — этот документ.

## Запуск

1. Откройте файл `index.html` в любом веб-браузере.
2. Нажмите на переключатель темы для перехода между светлой и тёмной схемой.
3. Используйте кнопки `ЧИТАТЬ ДАЛЕЕ` для перехода на дополнительные страницы.

## Как загрузить на GitLab

Для загрузки проекта на GitLab нужно выполнить следующие шаги:

1. Создать пустой репозиторий на GitLab.
2. Добавить удалённый репозиторий в локальный проект:
   ```bash
   git remote add origin <URL_вашего_GitLab_репозитория>
   
   ```
